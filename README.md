## 🌟 About This Project

This is a **production-grade job portal** built for **Shnoor International LLC** as part of a Software Engineer internship. It goes beyond a basic CRUD app — featuring an AI-driven interview system, TensorFlow.js-based proctoring, ATS resume scoring, and a three-role access system.

> Built with: React · Node.js · PostgreSQL · TailwindCSS · Socket.io · OpenRouter (GPT-4o-mini) · TensorFlow.js
